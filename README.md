# AIS C Walking CPR Tool

Standalone HTML clinical prediction tool for AIS C spinal cord injury that estimates walking recovery likelihood from exam inputs for clinician and research use.

## Live Deployment

This repository is configured for static hosting on Netlify.

- Entry file: `index.html`
- Netlify publish directory: project root (`.`)

## Local Files

- `index.html` — deploy entry page
- `ais-c-walking-cpr-refined.html` — original working file
- `netlify.toml` — Netlify deploy configuration
